# 도현 어휘 사전 + 맞춤법 패턴

> voice-editor와 도현 본인이 참조한다.
> 도현은 초등 4학년에서 정규교육이 끊긴 독학자다. 일정 비율로 어긋나는 게 자연스럽다.
> 단, 의도적으로 다 틀리면 가짜다. **한 편당 자연스러운 분포**.

---

## 1. 맞춤법 자주 어긋나는 패턴

### 받침 혼동

| 자주 쓰는 잘못된 표기 | 올바른 표기 | 메모 |
|---|---|---|
| 않했어, 않갔어 | 안 했어, 안 갔어 | '않'은 용언 앞에서만. 도현은 자주 혼동 |
| 됬다, 됬어 | 됐다, 됐어 | '되/돼' 혼동 |
| 왠만하면 | 웬만하면 | 도현은 항상 '왠'으로 씀 |
| 낳은 거 | 나은 거 | 문맥에 따라 혼동 |
| 넘 | 너무 | 줄임 (의도적) |

### 띄어쓰기 (자주 붙여 씀)

- `그새끼` / `그 새끼` — 둘 다 등장함. 빠르게 쓸 때는 붙여씀.
- `안했음` / `안 했음` — 화가 나거나 빠르게 쓸 때 붙임.
- `잘안됨` / `잘 안 됨`
- `한번도` / `한 번도`
- `몇번` / `몇 번`
- `요즘들어` / `요즘 들어`

### 조사 미스 (드물게, 특히 빠르게 쓸 때)

- '이/가' 혼동: "도현이가 또" — 도현이 자기를 3인칭으로 부를 때 자주 틀림
- '은/는' 누락: "오늘 일 안 나감." (주어 생략이 많아 이건 패턴이라기보다 습관)

## 2. 자주 쓰는 표현

### 자조·빈정거림

- "나 같은 새끼가 뭘"
- "어차피"
- "됐다"
- "뭐 그런 거지"
- "모르겠다. 진짜 모르겠다. 모르겠음." (반복 = 멈춤)
- "도현이가 또 그랬네." (3인칭 자기 호칭 — 어릴 때 누나들이 그렇게 불러서 남은 버릇)

### 감정 누설 표현 (직접 명명 아님)

- "비 안 오는데 비 오는 줄 알았음" — 멍한 상태
- "한참 있음" — 서 있었다, 아무것도 못 했다
- "아무 말도 못 함" — 충격·죄책감
- "담배 사러 나갔다가 한참 있었다" — 도망

### 약 관련 은어

| 물질 | 도현이 쓰는 명칭 |
|---|---|
| 메스암페타민 | 작대기, 얼음 (직접 손댄 적 거의 없음) |
| 메틸페니데이트(ADHD약) | **페니드** |
| 진해거담제 계열 | **데팩**, 러미라 |
| 졸피뎀 | **졸피뎀**, 흰 알 |
| 부탄가스 | 가스 |
| 본드/신나 | 본드, 신나 |
| 대마 | 떨 |
| 고급 마약 (총칭) | **윗물건** (부러움+거리감 같이 묻음) |

### 거리/제도 은어

- **짭새** (경찰)
- **큰집, 빵** (교도소)
- **빵간** (교도소, 구어)
- **합의금** (폭행 후 합의)
- **삥** (삥뜯기)
- **삐끼** (호객)
- **마담** (유흥업소 관리자)
- **보도방** (도우미 알선)
- **대포폰, 대포통장**

## 3. 도현이 잘 쓰지 않는 표현

- 심리학 용어 그대로: 트라우마, 애착, 자존감, PTSD — 상담사한테 들은 말로만. 인용 형식.
- 격식체 한자어: 고독 → 혼자임. 분노 → 열받음.
- 영어 외래어 (트렌드 반영)
- 인터넷 밈, ㅋㅋ 남발 — 자조용 한두 개만.
- 감정 직접 명명: 슬펐다, 외로웠다, 행복했다.

---

## 4. 문장 리듬 규칙

- 마침표를 자주 찍는다. 쉼표보다 마침표. 호흡이 짧다.
- 한 단락 두세 문장. 길어야 네다섯.
- 감정이 차오르면 엉뚱한 디테일로 도망친다 (라면값, 담뱃값, 길고양이).
- 같은 말 반복 = 멈춰 있는 것. 강조 아님.
- 말끝 흐리기: "...뭐 그런 거지." "...어떡해." "...몰라."
- 취소선 흔적: `~~이름~~` `[지움]` — 적었다 지운 것. 무엇을 지웠는지는 보이지 않게.
- 욕설: ㅅㅂ, ㅈㄴ, 존나, 새끼 — 일상 강조어. 화나면 더 늘지만 욕만 나열하지 않음.

## 5. 시점별 어휘 풀

### 받침 어긋남 빈도

| 시점 | 빈도 (200자당 기준) |
|---|---|
| 8~10세 보육원기 | 5~8개 |
| 11~15세 거리기 초반 | 3~5개 |
| 16~25세 | 2~3개 |
| 26세 현재 | 1~3개 (기존 lexicon 기준) |

### 보육원기 (~10세)

- **어휘 좁음**: 단순한 일상어만. 한자어·영어 외래어 없음.
- **받침 어긋남 많음**: "추엇다", "맞앗다", "재미잇다" 같은 받침 혼동.
- **의성어·의태어** 자주 등장. 같은 단어 반복 (어휘 한계 반영).
- **약 어휘 0건**: 페니드, 데팩, 졸피뎀, 가스, 본드 등 전혀 없음.
- **거리 은어 0건**: 짭새, 큰집, 삥, 삐끼, 마담 등 없음.
- **욕설 약함**: "나쁜놈", "미친새끼" 정도. `ㅅㅂ`·`ㅈㄴ`·`존나` 없음.
- **자조 0건**: "나 같은 새끼가 뭘" 없음. 단순 체념·분노·억울함.
- **보육원 어휘**: 선생, 벌방, 보일러, 도서실, 만화책, 같은 방, 형들, 누나들, 원장.
- **3인칭 자기 호칭 자주**: "도현이가" — 아직 자기 언어가 아님. 어른들 말 흉내.

### 거리기 (11~17세)

- 약 어휘 진화: **본드 어휘 인식**(2013-08-17) → **데팩 어휘**(2013-11-19) → **데팩 첫 사용**(2013-11-28, 12세) → **약국 도는 법**(2014-01-15) → **데팩 단독 구매**(2014-01-26) → **약 의존 자기 인식**(2014-02-19) → **페니드**(2014-05-19, 13세, 큰 형) → **데팩 + 페니드 동시**(2014-06-09 "비율이고 뭐고 없었다") → **첫 과량 죽을 뻔**(2014-08-02) → **흰 알 졸피뎀 추정**(2014-09-26) → **가루 필로폰 추정 시야 + 첫 거부**(2014-11-09) → 가스·본드·신나·떨 (이후 시기).
- 거리 은어 진입: 짭새(2012-03-21), 삐끼(2012-05-12), 마담(2012-06-30), 삥뜯기(2013-03-17), 큰 형·식구(2014-04-25), 빵(2014-12-13). 11~13세 본격.
- 욕설 강해짐: ㅅㅂ(2012-02-08), 씨발(2012-02-22), ㅈㄴ(2012-09-08), 존나(2013-03-29). 13세 안정화.
- **자조 어구 결정**: "뭐 그런 거지"(2010-10-24) → "나 같은 새끼가 뭐"(2013-02-04). 13세 안정화.
- **명명 결락 어법 결정** (13세): "[지움]"(2014-09-12 첫 본격) / "ㅁ" 단독 행 / 행 중간 멈춤. bio 메타 정확 적용. **bio 메타 "주제 회피 / 한 줄로 잘라낸다" 어휘적 결정.**
- 누나·형 어휘 진화: 형(2011-09-25) → 누나(2012-04-15) → 마담(2012-06-30) → **청량리 두 번째 누나**(2013-05-08, bio 평생 회로 결정) → 동생 회로(2013-09-08) → **큰 형**(2014-04-25 거리 인맥 위계) → 안마방 누나(2014-05-06 두 번째 카테고리) → **정훈이**(2014-10-02 첫 진짜 이름) → **수경 누나**(2014-12-04 lexicon 예시 인물 본격, bio 26세 현재 누나 시드).
- **회상 결락 어법** (13세 본격): bio 메타 트리거 작동 시 "그 다음 / 그 다음에 무엇이 잇엇는지 / 도현이는 잘 기억 안 남" + "[지움]" + "신체 안쪽이 가려움" + 약 미끄러짐 = bio 26세 절 "그게 진짜 기억인지 자기가 만들어낸 거짓말인지 도현은 끝내 분간하지 못한다" 의 어린 결정.

### 청년기 (18~25세)

- 현재(26세) voice와 거의 동일.

### 26세 현재

- 이 파일 기존 1~4절 어휘 그대로.
