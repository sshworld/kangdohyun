reality-checker를 단독으로 호출해 현실 디테일을 검증한다.

## 사용 방법

```
/verify 페니드 2026년 현재 법적 분류
/verify 청량리 588 현재 상태
/verify 고시원 서울 외곽 월세 시세
/verify 향정신성 의약품 단순소지 양형
```

## 절차

1. `reality-anchor` 스킬 로드.
2. **reality-checker** 에이전트 호출 — WebSearch/WebFetch 사용.
3. 팩트 카드 출력:
   - 사실 (2026년 기준)
   - 출처
   - 도현이 입에 올릴 법한 표현
