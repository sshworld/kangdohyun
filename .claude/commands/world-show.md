현재 세계관·인물·관계도를 한눈에 보여준다.

## 절차

1. `world/setting.md` 읽기 — 핵심 한 줄 요약.
2. `world/characters.md` 읽기 — 현재 등록 인물 목록.
3. `world/relations.md` 읽기 — mermaid 관계도 전체 출력.
4. `world/lexicon.md` 읽기 — 자주 쓰는 어휘 요약.

## 출력 형식

```
=== 도현의 세계 현황 ===

[배경]
(setting.md 핵심 1~2줄)

[등록 인물]
- 이름: 관계 / 상태
- ...

[관계도]
(relations.md mermaid 블록 전체)

[어휘 메모]
(lexicon.md 주요 패턴 요약)
```
