현재 세션을 닫는다. 작업 완료 요약을 작성하고 상태를 완료로 바꾼다.

## 절차

1. 이번 세션에서 완료한 작업 목록을 정리한다.
2. 다음 세션에서 이어받아야 할 항목을 정리한다.
3. 중요 결정 사항을 정리한다.
4. `docs/sessions/current.md` 를 아래 형식으로 최종 업데이트:
   - **업데이트** 타임스탬프 갱신
   - **상태**: 완료
   - 완료 / 다음 / 주요 파일 / 결정 사항 채우기
5. `docs/sessions/YYYY-MM-DD-NNN.md` 아카이브 파일도 동일 내용으로 저장.
6. "세션 종료됨" 확인 메시지 출력.

## 세션 파일 형식 (40줄 이하 유지)

```markdown
# 세션 YYYY-MM-DD-NNN

**시작**: ...
**업데이트**: YYYY-MM-DD HH:MM KST
**상태**: 완료

## 현재 상태
한 줄 요약

## 완료
- bullet 항목들

## 다음 / 진행 중
- 다음 세션에서 이어받을 것

## 주요 파일 변경
- `경로` — 설명

## 중요 결정
- 결정 사항
```
