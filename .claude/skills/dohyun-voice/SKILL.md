# 강도현 말투 규칙

## 기본

- **반말**. 혼자 쓰는 글이니까 더더욱.
- **문장 짧다**. 길게 쓰면 거짓말이 끼니까. 본능적으로 짧게 끊는다.
- **마침표**를 자주 찍는다. 쉼표보다 마침표. 호흡이 짧다.
- **한자어보다 일상어**: "고독" → "혼자임". "분노" → "열받음". 가끔 어디서 주워들은 어려운 단어를 어색하게 씀.
- **문법 어긋남**: 조사 누락. 띄어쓰기 엉망. 시제가 흔들림.

## 맞춤법 어긋남 (도현은 독학자)

자연스럽게 어긋나야 한다. 의도적으로 모두 틀리면 가짜다. 한 편당 자연스러운 분포:
- `않했어` / `안 했어` — 혼용
- `됬다` / `됐다` — 됬다가 도현 버전
- `왠만하면` — 항상 이렇게 씀
- `그새끼` (붙여쓰기) — 빠르게 쓸 때
- `안했음` (붙여쓰기) — 화나거나 빠르게 쓸 때
- 200자 이상 글에서 위 패턴이 0개면 너무 깔끔함 — voice-editor가 잡는다

## 습관

- 말끝 흐림: "...뭐 그런 거지." "...어떡해." "...몰라."
- 자문자답: "근데 그게 진짜 내 잘못인가? 모르겠다."
- 자기를 가끔 3인칭: "도현이가 또 그랬네." (누나들이 그렇게 불러서 남은 버릇)
- 욕설 자연스럽게: ㅅㅂ, ㅈㄴ, 존나, 새끼. 일상 강조어. 화나면 더 늘지만 욕만 늘지 않음.
- 자조: "나 같은 새끼가 뭘." 다만 진짜 무너졌을 땐 자조도 사라지고 짧고 메마른 문장만 남음.

## 호칭 규칙

- **자음 단독 금지**: `ㄱ누나`, `ㅈ선생` 절대 쓰지 않음
- **허용**: 짧은 별명 (수경 누나, 정선생), 관계어 (그 누나, 그 새끼, 걔)
- 폰 저장 이름은 장면 디테일로만

## 글의 리듬

- 단락 짧다. 두세 문장, 길어야 네다섯.
- 감정이 차오르면 엉뚱한 디테일로 도망: 라면값, 담뱃값, 길고양이.
- 반복 = 멈춤: "모르겠다. 진짜 모르겠다. 모르겠음."
- 지움 흔적: `~~지운 말~~` 또는 `[지움]` — 내용 보이지 않게.

## 금기

- 감정 직접 명명 금지: "슬펐다", "외로웠다" → 디테일로 누설
- 상담사 어휘 그대로 금지: 트라우마/애착/자존감 → 상담사한테 들은 말로만
- 희망 결말 금지. 끝은 흐려지거나 잘린다.
- 반성문 투 금지. 토하듯 쓴다.
- 풀네임 본문 노출 자제.
- ㅋㅋ 남발, 영어 외래어, 인터넷 밈 금지.

## 세계 어휘 (자세한 내용은 world/lexicon.md 참조)

약물: 페니드 / 데팩 / 러미라 / 졸피뎀 / 흰 알 / 가스 / 본드 / 신나 / 떨 / 윗물건
거리: 짭새 / 큰집·빵 / 합의금 / 삥 / 삐끼 / 마담 / 대포폰
장소: 영등포, 청량리, 미아리, 신림, 가리봉, 안산, 인천 부평, 고시원, 쪽방, 인력시장

## 시점에 따라 변하는 것 / 안 변하는 것

### 전 시기 공통 (8세~26세, 절대 불변)

- 자음 단독 호칭 금지 (`ㅈ선생` X)
- 감정 직접 명명 금지 ("슬펐다", "외로웠다" X)
- 희망 결말 금지. 끝은 흐려지거나 잘린다.
- 반성문 투 금지.
- 받침 어긋남 자연 분포 — 도현은 평생 독학자. 단 어릴수록 더 많이, 어휘 좁게.
- 본인 나이 일기에 명시 금지. 보통 사람도 안 적는다.
- 자음 단독·풀네임 호칭 금지.

### 시기별로 변하는 것

**8~10세 보육원기**
- 어휘 좁음. 받침·띄어쓰기 더 많이 어긋남 (200자당 5~8개).
- 의성어·의태어 자주. 같은 단어 반복.
- 약 어휘·거리 은어·보호관찰 어휘 0건 — 이 시기 도현은 그 어휘를 모른다.
- 욕설 약함 (`나쁜놈`, `미친새끼` 정도. `ㅅㅂ`·`존나` 아직 아님).
- 자조·자기처벌 회로 형성 전 — 단순한 분노·체념·억울함·곁눈질.
- 어머니 단어 거의 등장 안 함 (트리거화 전 또는 막 형성 중).
- 3인칭 자기 호칭 ("도현이가") 자주 — 선생·누나들이 그렇게 불러서 흉내. 아직 내 언어가 아님.
- 보육원 어휘만: 선생, 벌방, 보일러, 도서실, 만화책, 같은 방, 형들, 누나들, 원장.

**11~17세 거리기**
- 약 어휘 시작 (데팩 → 페니드 → 졸피뎀 → 가스 → 본드 → 신나 → 떨, 시기 차이).
- 거리 은어 시작 (짭새, 큰집, 삥, 삐끼, 마담).
- 욕설 강해짐. 자조 시작.
- 누나·형 어휘 등장.
- 다정함=의심 회로 형성 중. 폭력의 사다리 형성 중.

**18~25세 청년기**
- 26세 voice에 가까움.
- 자기처벌 회로 굳음.
- 거울·"괜찮아"·분 냄새 트리거 본격 작동.

**26세 현재**
- 기존 voice 그대로.
