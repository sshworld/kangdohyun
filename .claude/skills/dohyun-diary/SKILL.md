# 도현의 일기 메커니즘

## 형식

- **날짜**: 본문에는 흐리게. "오늘" "어제" "작년 봄" "2005년 어디쯤" 또는 안 적음. 정확한 날짜·나이 명시 금지.
- **길이**: 들쭉날쭉. 한 줄부터 한 페이지 이상까지.
  - 짧은 날: 1~3줄.
  - 보통 날: 5~15줄.
  - 폭주하는 날: 한 페이지 이상. 약을 했을 가능성 높음. 글이 어지럽고 시제 섞임.
- **지움 흔적**: `~~지운 말~~` 또는 `[지움]`. 무엇을 지웠는지 내용은 보이지 않게.

## 매체

일기가 적히는 매체는 시점에 따라 다르다:
- 보육원기(~10세): 도서실에서 주운 빈 페이지, 만화책 여백, 종이쪽지, 전단지 뒷면
- 거리기(10~17세): PC방 메모지, 담뱃갑 뒷면, 노숙 중 줍는 종이
- 청소년기 후반 이후: 폰 메모장 (대포폰 바꿀 때마다 흩어짐)

본문에 자연스럽게 새어나오면 좋음 (강제 X). "이거 도서실에서 주운 노트", "폰 새로 산 거에 옮겨 적음" 같은 한 줄.

## 빈도

주 2~5회. 하루 한 번 쓰는 강박 없음. 며칠 비우는 게 자연스럽다. 일기 누적 시 날짜 간격이 들쭉날쭉.

## 내용 패턴

- **표면**: 그날 있었던 일 — 그 시점 도현의 일상(보육원이면 식사·선생·벌방, 거리면 일용직·누나·약, 현재면 고시원·누나·약).
- **곁가지**: 핵심에서 도망치는 방식 — 보육원기는 만화책 디테일, 길고양이 본 거, 밥 이야기. 현재는 라면값, 담뱃값.
- **누설**: 곁가지에서 감정이 새어나온다. 도현은 그걸 자각하지 못한 척.
- **자기처벌**: "나 같은 새끼가", "어차피", "됐다" — 산발적으로. 보육원기엔 자기처벌 회로 미형성, 단순 체념·분노·억울함.

## 약을 한 날의 글

**도현이 약을 시작한 시점 이후만 적용 (거리 시기 이후). 보육원기(~10세) 일기에는 이 패턴 없음.**

- 더 짧아지거나 반대로 통제 없이 길어짐
- 같은 문장 반복
- 시제 무너짐 (과거가 현재로 미끄러짐)
- 갑자기 보육원·영등포 첫날 밤으로 날아감
- 다음 날: "어제 뭐 적었는지 기억 안 남"

## 회상 일기

그 시점 도현이 자기보다 어린 시절을 회상하는 일기도 있다(예: 17세 도현이 8세 보육원기를 떠올림). 이건 **회상하는 시점의 도현이, 그 시점의 어휘로** 쓴다.
- 17세 도현이 8세를 회상해도 일기 자체는 17세 어조로. 회상 속에 8세 어조가 섞이지 않음.
- "옛날에", "어릴 때", "그때" 식 표현으로 자연스럽게.

## 저장

`content/diary/` 폴더에 `YYYY-MM-DD-간단한제목.md` 형식으로 저장.
같은 날 여러 편이면 제목을 다르게 하거나 `-01`/`-02` suffix.
파일명 날짜는 정확한 가짜 날짜(시점 저장용). 도현 나이 = 그 해 - 2001.

## 예시 패턴

보육원기 짧은 날 (~8세):
```
모름

추엇다.
또 선생한테 맞앗다.
도현이가 왜 맞은 건지 모름.
그냥.
```

보육원기 보통 날 (~9세):
```
비

보일러 또 안됨. 다들 추워서 붙어 잠.
도현이는 혼자 도서실.
만화책 진짜 재미잇다. 그 사람 또 이겼음.
됐다.
```

짧은 날 (거리기 이후):
```
4/?? 수

비. 일 안 나감.
라면 두 개 끓여서 하나는 안 먹음.
됐다.
```

보통 날 (현재):
```
4/15 인 듯

누나가 또 약 끊으래.
자기는 술 먹으면서.
그 말 하는데 손목 멍이 아직 있더라. 내가 그랬지. 알아.
미안하다고 말 못 함. 그 말 하면 진짜로 미안한 게 되니까.
담배 사러 나갔다가 길에서 한참 있음. 비 안 오는데 비 오는 줄 알았음.
```

약한 날 (거리기 이후):
```
새벽 ?시

페니드 두 알 데팩 한 통.
머리 안 빈다 안 빈다 안 빈다.
옛날에 보육원에서 정선생이 내 머리를 책상에 박았는데 그게 왜 지금 생각나냐.
도현이가 그때 울었나. 모르겠다. 안 울었던 거 같다. 운 거 같기도.
~~누나~~ [지움]
그만.
```

무너지기 직전:
```
4/??

거울 봄.
안 봐야 했다.
```
