# 도현의 가사·시 메커니즘

## 핵심 원칙

도현의 가사·시는 **이미지·냄새·소리**로만 말한다.
감정을 직접 명명하는 것은 산문에서도 금지이지만, 시에서는 더 엄격하다.

## 형식

- **길이**: 짧게. 가사라면 한 곡 분량(절+후렴, 또는 절만). 시라면 10~20줄.
- **행 구조**: 짧은 행. 끊어지는 리듬. 1행이 1~5음절도 된다.
- **반복**: 후렴 반복 가능. 핵심 이미지의 반복도 가능. 단, 감정 명명의 반복은 안 됨.
- **여백**: 침묵이 중요하다. 적게 쓸수록 더 많이 느껴지는 경우가 있음.

## 도현의 시적 재료

| 재료 | 예시 |
|---|---|
| 냄새 | 분 냄새 / 소주 / 담배 / 라면 / 비 맞은 인도 |
| 촉각 | 손목 멍 / 벽을 치는 주먹 / 머리를 쓰다듬는 손 / 차가운 바닥 |
| 소리 | 창문 없는 방의 침묵 / 옆방 숨소리 / 역사 방송 / 새벽 버스 |
| 빛과 어둠 | 창문 없는 방의 낮밤 구분 못 함 / PC방 파란 빛 / 꺼진 핸드폰 |
| 행동 | 라면 두 개 끓이기 / 하나 안 먹기 / 담배 사러 나가기 / 거울 피하기 |

## 구조 패턴

- 이미지 나열 → 갑작스러운 끊김
- 반복 → 한 줄 돌변
- 질문 → 답 없이 끝

## 저장

`content/lyric/` 폴더에 저장.
