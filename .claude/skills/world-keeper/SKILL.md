# 세계 유지 규약

## 인물 사전 (`world/characters.md`) 항목 형식

```markdown
### (호칭/별명)
- **호칭**: (도현이 부르는 방식)
- **도현이 아는 것**: (한두 줄. 도현 시점만.)
- **도현이 느끼는 것**: (한 줄 선택)
- **상태**: 과거 / 현재 진행 / 소멸
- **첫 등장**: (파일명)
```

## 관계도 (`world/relations.md`) mermaid 갱신 규약

### 기본 그래프 구조

```mermaid
graph TD
  도현["강도현 (26세)"]
  (새 인물 노드명)["(별명/호칭) - 한 줄 메모"]
  도현 --"관계 레이블"--> (새 인물 노드명)
```

### 노드 이름 규칙

- 노드 ID: 영문 소문자 + 한글 허용. 공백 없음. (예: `현재누나`, `정선생`)
- 노드 레이블 `["..."]`: 별명/호칭 + 짧은 메모.
- **풀네임 금지**: 노드에도 풀네임 쓰지 않음.

### 연결선 레이블

- 관계 설명: 짧게. 단, 도현식 톤. ("멍 / 침묵", "라면·약·손찌검", "들러붙음 → 파괴")

### 클래스 스타일

```
classDef 부재 fill:#2a2a2a,stroke:#555,color:#aaa
classDef 현재 fill:#1a1a2e,stroke:#4a4a8a,color:#ccc
classDef 유동 fill:#1a1a1a,stroke:#333,color:#888
```

- `부재`: 어머니, 아버지 (사라진 관계)
- `현재`: 현재 진행 중인 관계
- `유동`: 오가는 관계 (과거 누나들, 동생들, 거리 형들)

## 호칭 규칙 (world-keeper가 항상 확인)

| 금지 | 허용 | 비고 |
|---|---|---|
| `ㄱ누나`, `ㅈ선생` 등 자음 단독 | `수경 누나`, `정선생` 짧은 별명 | 음절 가진 이름 |
| 풀네임 본문 노출 | `그 누나`, `그 새끼`, `걔` 관계어 | 익명성 유지 |
| 폰 저장 이름을 호칭으로 쓰기 | 장면 디테일로만 | "폰에 뭐라고 저장했는지도 모름" |
