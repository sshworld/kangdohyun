# 현실성 앵커 — 검증 트리거 가이드

## 검증이 필요한 키워드 분류

### 약물 (등장 시 무조건 검증)

- `페니드`, `메틸페니데이트`
- `졸피뎀`, `흰 알`
- `데팩`, `러미라`, 진해거담제
- `부탄가스`, `본드`, `신나`
- `떨`, `대마`
- `작대기`, `얼음` (메스암페타민 은어)
- `윗물건` (상급 마약 통칭)
- 약 복용량, 복용 방식 묘사

### 장소 (현재 상태 확인 필요)

- `청량리`, `588`
- `미아리텍사스`
- `영등포역`, `영등포`
- `고시원` (월세 시세)
- `인력시장` (일당 시세)

### 시세 (구체적 금액 등장 시)

- 고시원 월세 금액
- 인력시장·택배 상하차 일당
- 라면값, 담뱃값
- 합의금 규모

### 법·제도 (구체적 서술 시)

- 보호관찰 조건·기간
- 구치소 vs 교도소 호칭 사용
- 소년부 vs 형사부 구분
- 특정 죄목의 형량 언급
- 사설 보육원 시스템

---

## 합평 패널에서 reality-checker 호출 방식

합평 패널(panel-review) 단계에서 reality-checker는 자동으로 위 키워드를 확인.
검증할 내용이 없으면: "이번 글에 검증 필요 항목 없음. 이상 없음."

## /verify 수동 호출 시

`/verify <내용>` 입력 시 reality-checker를 단독으로 호출.
예: `/verify 페니드 2026년 현재 법적 분류`

---

## 원칙

- 검증은 **작가(사용자)를 위한 것**이지, 도현이 정확하게 쓰도록 강제하는 것이 아니다.
- 도현이 약물명을 뭉개거나 시세를 대충 말하는 것은 캐릭터상 자연스럽다.
- 사실 오류가 **명백하고 치명적인 경우**만 수정 권장. 미묘한 차이는 허용 범위.
