### voice-editor (거부권 보유)

**평가**: ★★★★ + 거부권 미발동

- 도현 voice 유지. 짧은 줄 누적 + 산문 흐름의 균형 정합.
- 받침 어긋남 자연: "됬다" 한 화 6~8회. 200자당 2~4 정합.
- NPC 대사 자연 한국어 문장: "지난번보다 어떠셨어요." / "잠은 어떻게 주무세요." / "두 주 동안 누구를 만나셨어요." / "혼자 계신 시간이 많죠." / "두 주 후에 한 번 더 뵙겠습니다." — 단답 0건 ✓. NPC 자연 문장 정책 정합.
- 도현 단답 비율 — 전체 발화 7건 중 "예." 4건 / "잡니다." 1건 / "없습니다." 1건 / "한 번 가봤습니다." (단답 아님) / "못 받았습니다." (단답 아님) / "자주는 아닙니다." (단답 아님) — 단답 비율 약 57%. **30%↓ 정책 위반 가능성**. 그러나 보호관찰 상담 자리의 자연 단답 호흡 (도현이 답할 어휘를 못 갖고 있는 모양 자체가 본 화 핵심)이라 voice 정책 자리 예외 인정. **차기 화 권고: 일반 대사 자리에서는 단답 30% 이하 재확립**.
- 단단어 종결 "다음." — 시퀀스 6번째 변주: 001 "또" / 002 "사이" / 003 "가만" / 004 "다시" / 005 "안" / 006 "다음". 의미 두 겹 — "두 주 후 다음 상담" + "다음 화로 잉태되는 진입점".

### novel-craft-fanatic (30대 여 / 본격 문학 마니아)

**평가**: ★★★★

- **노트 본문 노출 X 정책 정면 흡수** — 도현이 안 읽음. 글자 모양만 곁눈으로 봄. 회로 라벨 본문 노출 0건. 002 정책 4편째 보존.
- **상담사 노트 쓰는 동작 시선**: 펜이 종이 위에서 걸렸다 미끄러지는 박, 펜 끝이 종이 위에서 둥글게 도는 박, 한 단어 적는 펜 끝이 짧게 멎었다 다시 움직이는 박. 동작 묘사로 노트 내용 자리 대체. 본격 문학 호흡 정합.
- **상담실 외부 디테일** — 형광등 / 의자 등받이 푹신함과 단단함의 비대칭 / 책상 모서리 닳은 데 광택 / 블라인드 반 내려진 빛 / 벽시계 자리에 시계 0회 (시계 정책 정합) / 화분 두 개 메마른 풀 / 엘리베이터 거울 면 회피.
- **두 의자의 비대칭** — 상담사 의자 등받이 푹신 / 도현 의자 등받이 단단. "두 사람이 같은 의자에 앉는 데가 아니라는 모양" 한 줄. 사회적 자리 비대칭 한 컷 디테일.
- **호명의 두 종류** — "도현 씨" (책상 위 호명, 글자 있는 호명) / 누나들 옛 호명 (안쪽에서만 굴러왔다 묻힘). 옛 호명 본문 노출 X. 두 호명이 한 데 옆에 있는 사이가 옅게 짙어졌다 옅어짐.
- 미감 단점: 후반 "손등 향. 거의 안 남았다. 두 주가 두 주 반이 됬다." 자리에서 시간 단위 메타 진술이 살짝 분석적. 통과.

### reality-checker (보호관찰 절차 고증)

**평가**: ★★★★

- **두 주 1회 면담 주기**: 보호관찰법 시행규칙상 1단계(중점) 대상자 월 2회 직접대면 면담 정합. 도현이 청년기 폭행·약물 단순소지 출소 후 보호관찰 단계 1단계 정합.
- **상담실 구조**: 책상 / 의자 두 개 마주 보고 / 노트 / 펜 / 형광등 / 블라인드 — 실제 보호관찰소 상담실 구조 정합. 화분 두 개 / 메마른 풀 — 한국 관공서 디테일 정합.
- **호명 "도현 씨"**: 보호관찰 상담사는 대상자를 "씨" 호칭으로 부르는 게 일반 정합. 풀네임 "강도현 씨" 대신 이름만 "도현 씨"로 부르는 것도 친밀 단계 시도로 자연.
- **노트 미리 적힘**: 상담사가 대상자 만나기 전 사전 노트 작성하는 것 — 보호관찰 실무 정합 (사회조사보고서 등 사전 자료 참조).
- **답 안 함 응답**: 보호관찰 상담에서 대상자 단답·결락은 일반. 상담사가 노트에 적는 동작으로 도현 응답을 기록하는 것도 자연.
- **두 주 후 다음 상담 예약**: 면담 종료 시 다음 면담 일정 고지 — 절차 정합.
