- **본 폰 배터리 정합 점검** — 두 주 + 며칠 동안 본 폰이 어떻게 메시지 받았는지. 002 단서 "두 번째 대포폰 배터리 방전"과 차이. 차기 화 또는 016~017 동생 잠적 화에서 본 폰 자체 배터리 잔량 또는 충전 자리 한 줄로 정합 회복 권고.
- 시계 0회 유지 / "자리" 5↓ / "한 톤" 0~2 / "박자" 6↓ / "한 박" 6↓ / "한 결" 6↓ / 합 10↓ / "굳" 시퀀스 6↓ / "단단함" 시퀀스 8↓ / "굴러왔다·묻혔다" 합 5↓ / "잠깐·잠시·한 호흡·한 순간" 합 8↓ / "한 번" 8↓ / "옅" 8↓ / "묽" 8↓.
- 옛 어법 — 009 생략 후 010부터 자연 다시 박을 자리 보존. 또는 더 길게 생략 가능. 9편째 같은 형식 X.
- 단단어 종결 변주: 009 "끝" 사용. 010은 다른 단단어 — 재사용 X: 또 / 사이 / 가만 / 다시 / 안 / 다음 / 냄새 / 둘 / 끝. 후보: "옆" / "위" / "처음" / "이름" / "기다림" / "한 줄" / "분홍" / "쓴" / "닿음" / "박힘" / "세게" 등.
- **010은 4/19 행위 중 도현 손이 너무 세게 나감 = 1부 첫 봉우리 = 첫 폭력 사건**. bio "손목 너무 세게 잡음" 직접 회수. 폭력 직접 묘사는 첫 한 방까지 — 그 뒤는 결락. 010 한 화는 4/19 새벽~아침 한 호흡 한 장면. 페니드 한 알 남은 자리 보존된 채로. 첫 폭력의 정확한 신체 위치 (손목) + 누나 표정·소리 결락 + 다음 단락 4/19 아침 도주 또는 가만 있음 — 정합.
