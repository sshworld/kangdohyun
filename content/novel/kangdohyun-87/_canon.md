## 분량 협약

한 화 글자 수 약 7000~12000자. 산문 밀도 기준. 줄 수 아님.

## 어투 협약 (소설 본문)

- **시점 = 근접 3인칭**: 도현 = "도현은/도현이/도현". 누나 = "누나". "나" 1인칭 X. "도현이가" 자조 0~2회 한정.
- **대사 어법**: 뒤 말줄임표 ("잘 자.."). 앞 말줄임표 ("...잘 자.") 금지.
- **대사 흐름**: 도현 단답 30% 이하. NPC 단답 0건.
- **산문 호흡**: 문장 길이 다양. 한 단락 2~6문장. 한 줄짜리 단락은 강조·결락·임계에 한정.
- **단락 사이 빈 줄**: 장면 전환·시간 점프·강한 결락에만. 매 단락 사이 X.
- **묘사·행동 직조**: 공간·감각·내면이 함께 흐름.
- **일기와 구분**: 일기 패턴(한 줄 한 단어·매 줄 빈 줄)을 소설에 그대로 옮기지 않는다.

## 어휘 표기 정책

- **시간 단위**: 고유어 수사 + 의존명사로 **"두 주"** 통일. "이주" / "2주" 사용 X. ("한 주 / 두 주 / 세 주 / 보름 / 한 달" 풀)
- **거리·길이**: 정밀 측정 표시(도현 측정 강박 회로)에만 cm 사용. 1화는 "5cm" 첫 등장 + 좁아짐 자리 2회 한정. 나머지는 "한 손바닥 / 한 뼘 / 사이가 좁은 / 어깨가 닿을 듯한 / 살갗 사이 / 닿았다"로 자연 어휘.
- **"한 톤" 사용 범위 (강화 — 2026-05-12 003 합평 결정)**: 한 화 0~2회 한정. 색·명도·소리·온도 자연 자리만. 거리·다가옴·살갗·시간·동작·박자 X. 사용자 직접 지적: "자꾸 한 톤 한 톤 자연스러운지 판단" — **도배 어휘로 굳지 않게 다른 자연 어휘 풀로 분산**.
  - 허용: 색 (한 톤 어두워졌다), 소리·분위기 (한 톤 무거웠다), 온도 (한 톤 더 더웠고)
  - 금지: 거리 / 살갗 / 동작의 박자 / 시간 / 향이 옅어지는 단계 (이건 "옅어졌다 / 묽어졌다 / 가셨다"로)
  - **변주 풀**: 옅어졌다 / 짙어졌다 / 묽어졌다 / 풀렸다 / 가셨다 / 흩어졌다 / 가라앉았다 / 굳었다 / 시들었다 / 멎었다
- **"자리" 사용 범위 (신규 — 2026-05-12 003 합평 결정)**: 한 화 5회 이하. 추상 명사화 회피. 사용자 핵심 지적 흡수: "한 톤" 정리하니 "자리"가 새 도배 어휘로 굳음 (003 v1 본문 25회).
  - **변주 풀**: 데 / 곳 / 쪽 / 칸 / 모서리 / 안쪽 / 모양 / 가운데 / 사이 / 그대로 / 채로 / 안 / 안 자리 / 검정 / 길 / 동작
  - 일부는 그냥 삭제 가능 (불필요한 추상 명사화)
- **"박자" 사용 범위 (신규 — 2026-05-13 004 자체 적발)**: 한 화 6회 이하. 음악적 박자(beat·진동·일정 간격) 의미 자리에만. 사용자 본질 지적의 연장 — "자리" 정리하니 "박자"가 새 도배 어휘로 굳음 (004 v1 본문 57회).
  - **변주 풀**: 박 / 칸 / 결 / 번 / 흐름 / 모양 / 걸음 / 간격 / 감각 / 종류
  - 시간 흐름·인물 행동·신체 감각의 미세 단위는 "박자"로 굳지 않게 변주
  - 음악적 박자(콘크리트 상판 차 진동·계단 형광등 깜박임·옆방 사람 신발 박자) 자리만 보존
- **공식 회피 동작 어휘**: "한 번 끊었다 / 인식한 게 더 이상했다" 한 화 2회 한정. 같은 박자로 단락마다 박지 말 것. 변주 어휘: "그 자리를 두었다 / 그쪽도 두었다 / 손에서 놓았다 / 어긋났다 / 더 가지 않았다 / 흘려보냈다 / 가라앉기를 기다렸다"
- **자기 동작 분석**: 한 화 1~2회 한정. 도현이 자기 회로를 외부에서 보는 자리는 강조 모멘트로만. 같은 단락 안에 자기 분석 두 번 X.
- **신체 직설어**: "자지·보지" 거리 직설 풀 한정 허용. 도배 X (한 화 5회 이하). "음경·질·페니스" 같은 의학·해부학 어휘 절대 금지.
- **회피 회로 "거기"**: 도현 voice 핵심 보존. 직설어와 동거.

### 본격 문학 호흡 의식 (사용자 지적 흡수)

신경숙·황정은·하성란·한강의 근접 3인칭 호흡 기준. 같은 효과를 매번 다른 어휘로 친다. 도배가 voice가 아니라 **버릇**으로 굳지 않도록. 매 화 후 voice-editor + novel-craft가 정량 점검.
